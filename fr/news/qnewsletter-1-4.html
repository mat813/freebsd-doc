Volume #1 Numéro #4
Août 1998

  • Quelques informations sur les versions de FreeBSD : La dernière version de FreeBSD est la 2.2.7-RELEASE. FreeBSD 2.2.8 est prévu pour le 15 novembre. FreeBSD 3.0 est prévu pour le 15 octobre.

  • Annonce : OpenLDAP : Le project OpenLDAP est heureux de vous annoncer la disponibilité de la version 1.0 de OpenLDAP. Ce logiciel est disponible en téléchargement à : http://www.OpenLDAP.org/download.html ou par accès CVS anonyme à : http://www.OpenLDAP.org/repo.html

  • PicoBSD: PicoBSD 0.4 est disponible ! http://www.FreeBSD.org/~picobsd/ Pour le télécharger : http://www.FreeBSD.org/~picobsd/picobsd/

  • FreeBSD-Advocacy : Walnut Creek contre Microsoft. http://www.crl.com/wccdromrcd.html

  • FreeBSD : solide comme un roc. http://www.zdnet.co.uk/athome/misc/toot/output/57.html

  • Daemon News : le magazine électronique des BSD Les fans de BSD ont leur magazine. Rejoignez les pour écrire un article ou tenir une rubrique. http://ezine.FreeBSD.org La première édition sera disponible le 1er Septembre !!!! Voyez à : http://www.daemonnews.org

  • Produits commerciaux : "Citadel", un coupe-feu ("firewall") et routeur VPN basé sur FreeBSD. Plus de détails à http://www.cdsec.com

  • Equalizer V2.0 : Equalizer est un outil de répartition de charge et de gestion de cluster. http://www.coyotepoint.com/cp_products.shtml

  • FreeQualizer V2.0-b4 : FreeQualizer est une version de développement gratuite de Equalizer pour FreeBSD 2.2.6 http://www.coyotepoint.com/freequalizer.shtml

  • FreeBSD HOW-TO : En portugais : http://www.geocities.com/SiliconValley/Sector/8050/

  • FreeBSD pour les utilisateurs Linux : http://freebsd.stanford.edu/FreeBSD/linux.html

  • La sécurité sous FreeBSD : http://www.best.com/~jkb/howto.txt

  • Les pages de ressources sur UNIX : La page d'index des ressources pour UNIX : http://unix.cyberscript.net/ FreeBSD-Online : (Son addresse a changé !!! Voici la nouvelle adresse.) http://freebsd.unixhelp.org/
