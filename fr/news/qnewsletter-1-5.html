Des nouvelles du monde FreeBSD.
Volume #1 Numéro #5
Septembre 1998

  • Informations sur les versions de FreeBSD : La dernière version est la 2.2.7-RELEASE. FreeBSD 2.2.8 est prévu pour le 30 Novembre. FreeBSD 3.0 est prévu pour le 15 Octobre. Commencez les béta tests !!!

  • Article sur FreeBSD : http://www.zdnet.com/sr/stories/issue/0,4537,349576,00.html

  • Améliorations récentes : CAM ("Common Access Method"), couche SCSI implémentée dans la version 3.0-Current.
    Perl5 est désormais inclu en standard dans la version 3.0-Current
    Support des disques durs IDE d'une capacité supérieure à 8 Go dans la version 2.2
    La version 3.0-CURRENT est maintenant en ELF !
    FreeBSD pour Alpha est désormais autonome - le noyau et le "make world" fonctionnent !

  • Nouvelle version du JDK pour FreeBSD : Toutes les informations concernant son téléchargement sont sur : http://www.FreeBSD.org/java

  • Interface Uniformisée pour les Pilotes de périphériques ("Uniform Driver Interface" - UDI) : http://www.sco.com/udi ftp://ftp.cup.hp.com/pub/hp_stds/udi/docs.html http://www.infoworld.com/cgi-bin/displayStory.pl?980916.ecminer.htm http://www.intel.com/pressroom/archive/releases/Cn091698.htm ftp://ftp.cup.hp.com/pub/hp_stds/udi/Postscript/rules.ps.Z

  • Pilote pour carte Ethernet RealTek 8139 : des testeurs sont demandés. http://www.FreeBSD.org/~wpaul/RealTek/3.0 source pour FreeBSD 3.0 http://www.FreeBSD.org/~wpaul/RealTek/2.2 source pour FreeBSD 2.2.x Répondre à : wpaul@skynet.ctr.columbia.edu.

  • ThunderLAN Ethernet driver (popular in Compaq machines) : Testers Wanted. http://www.FreeBSD.org/~wpaul/ThunderLAN/3.0 source for FreeBSD 3.0 http://www.FreeBSD.org/~wpaul/ThunderLAN/2.2 source for FreeBSD 2.2.x Reply to: wpaul@skynet.ctr.columbia.edu.

  • Produits commerciaux pour FreeBSD : Emerging Technologies, Inc. Cartes ISA et PCI "Sync" pour FreeBSD http://www.etinc.com Gestionnaire de bande passante http://www.etinc.com/bwmgr.htm
    Acadix Software Systems : APE est un environnement de développement intégré pour Unix avec colorisation de la syntaxe, qui est indépendant du langage utilisé et indépendant du type de terminal. http://www.execpc.com/~acadix

  • Magazine en ligne sur les systèmes BSD : "Daemon News" ! N'oubliez pas de consulter le magazine en ligne sur les systèmes BSD réalisé par de nombreux membres de la communauté FreeBSD. Les articles doivent être envoyés avant le 15 de chaque mois et sont publiés le 1er du mois suivant. Tous les auteurs sont les bienvenus. http://www.daemonnews.org/

  • Citation : Si vous pointez un pistolet en direction de votre pied et que vous appuyez sur la détente, c'est le travail du système UNIX d'assurer une livraison fiable de la balle à l'endroit où vous avez pointé le pistolet (dans le cas présent, Monsieur Pied). (Terry)
