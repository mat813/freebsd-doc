Des nouvelles du monde FreeBSD.
Volume #2 Numéro #1
Janvier 1999

  • Informations sur les versions de FreeBSD : FreeBSD 2.2.X est désormais en mode maintenance (il n'y a plus de développement actif)
    FreeBSD 3.X est désormais la branche stable avec la version 3.1 prévue pour le 15 Février 1999.
    FreeBSD 4.0 est désormais la version de développement sans aucune date de sortie prévue pour le moment.

  • Nouvelles sur FreeBSD : Greg Lehey a écrit un article dans "Sunworld". http://www.sunworld.com/swol-01-1999/swol-01-bsd.html

    Sean Eric Fagan a écrit un article dans "LINUX JOURNAL" http://www.linuxjournal.com/issue57/2515.html

  • FreeBSD Advocacy : FreeBSD a désormais son propre magasine en ligne : http://www.freebsdzine.org

  • Liste des Groupes d'Utilisateurs de FreeBSD http://www.dsinw.com/~hamellr/freebsd/ug.html/

  • FreeBSD est cité dans http://helix.nature.com/webmatters/tomog.html

  • IPsec Le kit "IPsec" de KAME pour FreeBSD 3.0-RELEASE est disponible. Ce kit fourni une implémentation de "IPsec" pleinement fonctionnelle pour FreeBSD 3.0-RELEASE. N'hésitez pas à rapporter les éventuels bugs à l'équipe de KAME via la base de bugs GNATS de KAME (n'utilisez pas le GNAT de FreeBSD). Merci. ftp://ftp.kame.net/pub/kame/snap/kame-ipsec-19990104-fbsd30-snap.tgz

  • Les guides : Guide pour les cartes sons http://www.neosoft.com/~conrads/awepnp-freebsd.txt

  • Le bestiaire BSD : Le numéro de Janvier de "Daemon News" met en vedette un nouveau personnage de bande dessinée BSD. Le numéro de Février mettra en co-vedette un démon féminin. http://www.daemonnews.org/
