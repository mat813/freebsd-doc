  • Informations sur les versions de FreeBSD : FreeBSD 3.1 est disponible chez Walnut Creek CDROM
    FreeBSD 4.0 est désormais la version de développement sans aucune date de sortie prévue pour le moment.

  • Nouvelles sur FreeBSD : http://lwn.net/1999/features/ACInterview/ http://advisor.gartner.com/n_inbox/hotcontent/hc_2121999_3.html

  • FreeBSD Advocacy : Gateway Computers conçoit l'idée de livrer FreeBSD avec certains de leurs serveurs. Si vous souhaitez acheter un ordinateur Gateway avec FreeBSD, contactez : 
      	    Fritz Fitzgibbons 
	    fitzgfri@gateway.com
	    425.803.1947

  • Daemon News : Les articles peuvent maintenant être envoyés à l'adresse news@daemonnews.org Nous recherchons des articles concernant :

    Les questions liées à la licence BSD (GPL versus Artistic versus BSD)
    La sécurité / la détection des intrusions
    Que faire après un crash / L'analyse des "core dump"
    Les débuts des systèmes BSD.

    Si vous êtes intéressé pour écrire un article sur ces sujets, contactez: editors@daemonnews.org ou envoyez vos articles à : article@daemonnews.org

  • Netware: Un client FreeBSD pour netware en version béta est disponible ftp://ftp.butya.kz/pub/nwlib/ncplib.tar.gz

  • Ressources sur les OS : Une page web sur les systèmes d'exploitation http://www.skylab.org/~sabre/os/index.html

  • Gestionnaire d'administration système pour FreeBSD : Version 1.6 http://homepage.esoterica.pt/~jardim/Index.html
